## 📖 项目简介

**「智驭苍穹・守路安澜」** 是一款面向铁路行业的空天地多模态短临致灾风险孪生预警系统。项目融合 **AI、气象、铁路、低空** 四大技术领域，通过独创的 **LoongClaw 动态路由混合专家系统**，实现铁路沿线灾害风险的精准预警。

> 一句话定位：**用AI守护每一条铁路，让出行更加安全。**

### LoongClaw 引擎 — 四大专家

| 专家 | 核心技术 | 核心指标 |
|------|----------|----------|
| 🤖 **AI专家** | MambaSwin-UNet-STA 深度学习模型 | 命中率 0.82，45M 参数，200ms 推理 |
| 🔬 **物理专家** | COTREC + WRF 数值模式 | 物理约束强，可解释性高 |
| 🌤️ **气象专家** | FY-4A卫星 + 多普勒雷达 + 探空 | 多源数据融合，全域覆盖 |
| 🚄 **铁路专家** | 铁路致灾知识图谱 + 脆弱性曲线 | 行业知识驱动，精准评估 |

## 🚀 快速开始

### 环境要求
- **微信开发者工具** v1.06.2307260+
- **基础库版本** 2.32.0+
- **Node.js** 16+

### 本地运行
```bash
# 1. 克隆仓库
git clone https://github.com/NOBEL-Pxx/railway-ai-warning.git

# 2. 打开微信开发者工具

# 3. 导入项目，选择 miniprogram 目录

# 4. 在 project.config.json 中配置你的小程序 AppID

# 5. 点击编译运行
```

### 发布上线
1. 在微信开发者工具中上传代码
2. 登录 [微信公众平台](https://mp.weixin.qq.com/) 提交审核
3. 审核通过后发布
