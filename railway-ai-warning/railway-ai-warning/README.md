# 智驭苍穹・守路安澜 - 铁路防灾AI预警系统

## 项目简介

「智驭苍穹・守路安澜」是一款面向铁路行业的空天地多模态短临致灾风险孪生预警系统。系统融合AI、气象、铁路、低空四大技术领域，通过独创的LoongClaw动态路由混合专家系统，实现铁路沿线灾害风险的精准预警。

## 核心功能

- **项目展示**：系统定位、核心价值、应用场景
- **技术架构**：四层技术栈可视化展示
- **LoongClaw引擎**：四大专家模块详解
- **空天感知**：无人机集群平台展示
- **成果验证**：模型精度数据与实战案例
- **商业模式**：三级盈利火箭与市场分析
- **团队介绍**：跨学科尖兵团队展示
- **预约演示**：在线表单提交与联系方式

## 技术栈

- 框架：原生微信小程序
- 语言：WXML + WXSS + JavaScript
- UI设计：苹果风格，磨砂玻璃效果
- 图表：原生Canvas + F2
- 3D展示：Three.js小程序适配版

## 设计规范

### 色彩系统

- 主色：#165DFF（深空蓝）
- 辅助色：#7B61FF（科技紫）
- 成功色：#00CC99（安全绿）
- 警告色：#FF9500（警示橙）
- 危险色：#FF3B30（苹果红）

### 字体规范

- 大标题：40-56rpx，粗体
- 副标题：28-32rpx，中等
- 正文：26-28rpx，常规
- 辅助文字：20-24rpx

### 组件规范

- 卡片圆角：16-24rpx
- 按钮圆角：12-16rpx
- 阴影：0 4rpx 24rpx rgba(0,0,0,0.08)
- 磨砂玻璃：backdrop-filter: blur(40rpx)
