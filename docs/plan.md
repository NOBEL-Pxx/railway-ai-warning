# 项目计划

## 基本信息

- **项目名称**: 气候暖湿化背景下铁路沿线小流域短临降水智能预测与致灾风险推演
- **项目类型**: 大学生创新训练计划
- **执行周期**: 2026年4月 - 2027年4月
- **负责人**: 彭小溪
- **团队成员**: 文年平（信息科学与工程学院）
- **指导教师**: 胡淑娟（大气科学学院教授）

## 核心目标

1. **短临预报突破**: 实现0-3小时分钟级降水预报，TS评分较传统方法提升≥15%
2. **风险推演能力**: 建立从降水场到路基冲刷、边坡失稳等工程风险的定量映射
3. **成果产出**: 开源代码库、技术验证报告、核心期刊论文1-2篇

## 技术路线

### 核心技术
- OpenClaw动态路由混合智能框架
- MambaSwin-UNet-STA深度学习架构
- Y3垂起固定翼无人机机动观测
- SCS-CN水文模型耦合

### 数据资源
- 兰州铁路局20年雨量计数据（2004-2024）
- 中国气象数据网雷达拼图
- ERA5再分析资料
- NASA SRTM 30m DEM

## 阶段划分

| 阶段 | 时间 | 主要任务 |
|------|------|----------|
| 数据治理 | 2026.4-5 | 数据清洗、特征工程、基线验证 |
| 模型训练 | 2026.5-8 | 架构搭建、训练调优、消融实验 |
| 系统集成 | 2026.8-11 | 水文耦合、风险推演、接口开发 |
| 验证评估 | 2026.12-2027.1 | 个例验证、实地调研 |
| 结题验收 | 2027.2-4 | 论文撰写、代码整理、答辩 |
