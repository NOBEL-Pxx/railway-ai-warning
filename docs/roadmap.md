```

        │
        ▼

        │
        ▼
┌───────────────────┐
│   瓶颈层          │

        │
        ▼
┌───────────────────┐

        │
        ▼
```

## 关键技术点

### 1. Mamba模块
- 线性复杂度 O(n)
- 适合长序列时空建模
- 突破Transformer二次复杂度瓶颈

### 2. Swin Transformer
- 窗口注意力机制
- 层级特征融合
- 局部-全局特征提取

### 3. 时空注意力 (STA)
- 时间注意力: 捕捉降水演变趋势
- 空间注意力: 捕捉空间分布特征
- 联合建模: 时空动态特征

### 4. 物理约束
- 质量守恒
- 地形强迫
- 非负约束
