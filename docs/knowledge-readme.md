# 项目知识库

此目录存放项目相关的知识文档，供 LoongClaw 查询使用。

## 目录结构

```
knowledge/
├── project/           # 项目文档
│   ├── plan.md       # 项目计划
│   └── roadmap.md    # 技术路线
├── data/             # 数据文档
│   └── data_dict.md  # 数据字典
├── models/           # 模型文档
│   └── architecture.md  # 模型架构
└── experiments/      # 实验记录
    └── README.md     # 实验记录说明
```

## 使用方式

LoongClaw 会自动加载此目录下的知识文档，用于：
- 回答项目相关问题
- 提供技术决策参考
- 记录实验经验
